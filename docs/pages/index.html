Scaling Flow

Namespace Suffix Mode

Instead of listing namespaces manually, define a suffix and the agent automatically watches all matching namespaces:

orders-dev payments-dev users-dev orders-test payments-test orders-qa payments-qa orders-prod payments-prod users-prod 
apiVersion: kairos.maximilianopizarro.github.io/v1alpha1
kind: KairosAgent
metadata:
  name: agent-production
spec:
  mode: supervised
  watch:
    namespaceSuffix: "-prod"   # Watches ALL *-prod namespaces
    resourceTypes:
    - Deployment
    - StatefulSet

Quick Start

# 1. Install CRDs
kubectl apply -f config/crd/bases/

# 2. Create namespace and RBAC
kubectl apply -f config/samples/prereqs.yaml

# 3. Deploy the operator
make deploy IMG=quay.io/maximilianopizarro/kairos-operator:v0.1.0

# 4. Create a scaling policy
kubectl apply -f config/samples/kairos_v1alpha1_smartscalingpolicy.yaml

# 5. (Optional) Deploy AI agent
kubectl apply -f config/samples/kairos_v1alpha1_kairosagent.yaml

# 6. Deploy the console
kubectl apply -f config/samples/kairos_v1alpha1_kairosconsole.yaml
